# Email_marketing
 Add .env file of your own
## Inside .env
```
GOOGLE_API_KEY=""

EMAIL_ADDRESS=""

EMAIL_PASSWORD=""

YOUR_NAME=""
```

Note: If you are using Gmail with 2FA. Please follow as per this: [Here](https://support.google.com/mail/answer/185833?hl=en-GB)

## How to run:

In terminal run: `streamlit run Email_marketing.py`
